## 三、页面级验收建议

### 1. Home

- 查看课程验收面板是否显示真实数据行数、列数、模块状态、LLM 状态和测试命令

### 2. Data Pipeline

- 查看默认样例数据是否成功加载
- 查看原始预览、清洗预览、缺失值与清洗报告
- 上传一个通用 CSV，确认系统不崩溃

### 3. Dashboard

- 查看核心指标卡
- 查看发行趋势、类型分布、标签分布、价格分布和口碑分布
- 查看“数据发现”结论是否来自真实统计

### 4. Visual Explorer

- 修改筛选条件，确认图表和表格随之变化

### 5. Data Q&A

- 输入自然语言问题，确认回答引用当前数据而非空泛描述

### 6. Idea Lab

- 输入创意文本，查看创意解析、竞品、机会评分、差异化建议和简报

## 四、提交前最后检查

1. 运行 `python3 -m pytest -q`
2. 检查 README 和文档是否为中文且内容一致
3. 检查默认样例数据文件存在
4. 检查 PPT、演示视频和数据来源补充说明是否已准备
5. 检查答辩时是否能明确说明“LLM 不参与真实统计计算”

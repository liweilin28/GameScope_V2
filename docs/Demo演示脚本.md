# Demo 演示脚本

## 0:00-0:25 项目定位

说明：

我做的不是游戏，而是一个面向独立游戏开发者的 Steam 游戏市场智能分析平台。

展示：

Home 页面，说明 GameScope 的核心价值是把 Steam 历史数据转化为立项参考。

## 0:25-0:55 数据读取与清洗

展示：

Data Pipeline 页面。

说明：

系统支持默认数据和上传 CSV，并完成字段兼容、缺失值处理、评论数和好评率计算、年份提取、Indie 判断等预处理。

## 0:55-1:25 市场总览与可视化

展示：

Market Dashboard 页面。

说明：

这里展示发行趋势、类型分布、标签分布、价格分布、口碑分布等，可以快速理解 Steam 市场的基本结构。

## 1:25-1:55 Data Q&A

展示：

输入示例问题：

```text
Indie 游戏价格主要集中在哪？
```

说明：

系统不是让 AI 直接乱答，而是后端函数先计算结果，AI 或模板只负责解释。

## 1:55-2:45 Idea Lab

展示：

输入游戏创意：

```text
我想做一款 2D 独立解谜叙事游戏，价格控制在 20 元以内，风格偏治愈和剧情向。
```

展示内容：

- 创意解析
- 机会评分
- 竞品雷达
- 差异化建议
- Project Brief

说明：

Idea Lab 是 PLUS 创新功能，它把数据分析转化为早期游戏立项辅助。

## 2:45-3:00 总结

说明：

本项目完成了课程五个基础模块，并通过 PLUS 模块把 Steam 数据分析转化为面向独立游戏立项的决策辅助工具。AI 主要用于辅助开发、创意解析和自然语言报告，但所有关键计算由 Python 后端完成。

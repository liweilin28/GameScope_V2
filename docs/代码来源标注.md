# 代码来源标注

## 一、说明

本文件用于按课程提交要求，对项目主要代码与文档文件的来源进行透明标注。这里不对 AI 贡献做比例估算，只说明主要来源类型，以及学生是否进行了人工修改与验证。

来源标记说明：

- 学生：主要由学生编写
- AI：主要由 AI 生成初稿，后续由学生审核
- 学生 + AI：学生提出需求、组织结构并人工修改，AI 参与生成或补充部分内容

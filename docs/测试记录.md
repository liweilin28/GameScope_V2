# 测试记录

## 测试环境

- 操作系统：Windows
- Python：3.14
- 后端框架：FastAPI
- 前端：HTML/CSS/Vanilla JavaScript
- 测试工具：pytest

## 安装依赖命令

```bash
pip install -r requirements.txt
```

## 启动命令

```bash
python -m uvicorn backend.main:app --reload
```

## pytest 测试命令

```bash
pytest
```

如命令不可用：

```bash
python -m pytest
```

## 已知问题和解决方案

- 端口被占用：使用 `--port 8001`。
- 找不到默认数据：上传 CSV 或放置到 `data/sample/sample_steam_games.csv`。
- 上传字段不完整：系统会关闭部分分析能力并显示提示。
- LLM 未配置：使用 Rule-based fallback。
- `pytest` 命令不可用：使用 `python -m pytest`。
